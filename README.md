<!-- GETTING STARTED -->
## Getting Started

If you want to run the app on your machine follow the steps

### Prerequisites

* Git
* Node

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/bambolelooo/techBlog.git
   ```
2. Install NPM packages
   ```sh
   npm install
   ```
3. run the app 
   ```sh
   npm start
   ```
